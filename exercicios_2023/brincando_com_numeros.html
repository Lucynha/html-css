Usando Números no JavaScript

Números Inteiros

Para usar os números inteiros no JS, é necessário fazer a conversão de string para número, usando o seguinte comando: Numbe.parseInt(n)

Números Flutuantes

Para usar os números Flutuantes(números com vírgulas) no JS, é necessário fazer a conversão de string para número, usando o seguinte comando: Numbe.parseFloat(n)

Nesse exemplo, a letra n é usada para representar a variável escolhida.

Podemos usar também Number.(n) sozinho, dessa forma o JS vai diferenciar se o número é inteiro ou flutuante.
